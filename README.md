# Joyful Activities

A simple React application to help you reflect on joyful activities and find meaning in the little things.

## Features

- Guided self-reflection exercise
- Multi-step process to explore feelings and meanings
- Affirmation screen
- Built with React, TypeScript, and Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite

- React
- React Router
